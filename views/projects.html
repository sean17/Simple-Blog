I've been blessed with an abundance of opportunities in my life. Below are some of the projects I've been able to work on.

Cours.ly

Front-End Developer

January 2012 - Present

Cours.ly is a social educational platform that enables users to organize, engage and share. We are a company focused heavily on the end-user experience. Through Cours.ly I have been able to learn an incredible amount about javascript along with its many frameworks and platforms. Our front-end consists of javascript with Spinejs and our backend uses Ruby on Rails as well as Nodejs.

Impress.ly

Front-End Developer

March 2012 - Present

As a part of Cours.ly, our team decided to develop a useful slide creation tool called Impress.ly. The tool is used to author slides interpretted by Impressjs, as this feature currently does not exist for Impressjs. The project was largely developed during UBHacking 2012 and was strong enough to be the winning project.

Distributed Chat Messenger

Solo Developer

January 2012 - Present

Expanding my understanding of Sockets and Android programming, I have developed a simple chat messenger that allows for 5 Android emulators to chat amongst each other. This project has been useful as a proof of concept for Distributed Systems theories as well as a simple project in Android.

Wearable Track Pad

Solo Developer

May 2011

Computer Scientists typically avoid getting too close to hardware. The closest I've come is with Arduino, designing a trackpad as "proof of concept". I used conductive fabric to for the trackpad surface. I then used an Arduino Uno to trace finger location and display the result graphically in a visual program using Python.

UB Bus Tracking System

Project Leader

September 2011 - Present

Those of you that are familiar with the UB-Stampede are well aware that it is a spotty form of transportation between UB's North and South campuses at best. Along with a few others, I have been leading a group of undergraduates to design a system capable of tracking the busses and displaying their location to students. This system will be using Nexus S phones to track the busses and display the results using a google maps API.
